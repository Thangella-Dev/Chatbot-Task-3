VoiceFlow Assistant

A local-first chat widget with voice input, language labels, and smart answers.

What you can ask

  • Delivery types, items available, delivery hours
  • Order tracking and payment methods
  • General questions (only if AI fallback is enabled)

How to use

  1. Start the backend: npm start
  2. Click the chat button (bottom-right)
  3. Pick a suggestion chip or type your question

Notes

  • Runs locally on your machine
  • Mic works best on Chrome (may require internet)
  • Weather + optional AI require internet

How it works

Your message is shown instantly, then the backend detects language and returns an answer from the database (or other tools).
